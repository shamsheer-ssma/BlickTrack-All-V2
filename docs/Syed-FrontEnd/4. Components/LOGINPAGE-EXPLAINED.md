### **Line 1: Client Component Directive**

```typescript
'use client';
```

**What is this?**
- A **Next.js 13+ directive**
- Marks this component as **client-side only**

**Why needed?**
```typescript
// ‚ùå Without 'use client':
// Cannot use useState, useEffect, event handlers
// Component runs on SERVER only

// ‚úÖ With 'use client':
// Can use React hooks, browser APIs
// Component runs on CLIENT (browser)
```

**Server vs Client Components:**

| Feature | Server Component | Client Component ('use client') |
|---------|------------------|--------------------------------|
| **Hooks** | ‚ùå No (`useState`, `useEffect`) | ‚úÖ Yes |
| **Event handlers** | ‚ùå No (`onClick`, `onChange`) | ‚úÖ Yes |
| **Browser APIs** | ‚ùå No (`localStorage`, `window`) | ‚úÖ Yes |
| **Runs where** | Server only | Client (browser) |
| **Default in** | Next.js 13+ App Router | Must add `'use client'` |

**When to use `'use client'`:**
- ‚úÖ Forms with `useState`
- ‚úÖ Interactive components
- ‚úÖ Browser APIs (`localStorage`, `window`)
- ‚úÖ Event handlers (`onClick`, etc.)
- ‚úÖ Third-party libraries that use hooks

**What is `useEffect`?**
- Runs code **after** component renders
- For side effects (animations, API calls, subscriptions)

**Syntax:**
```typescript

  // Code to run
}, [dependencies])
```

**Dependencies array `[]`:**
- `[]` ‚Üí Run ONCE (on mount)
- `[value]` ‚Üí Run when `value` changes
- No array ‚Üí Run on EVERY render (usually bad!)

**Your code:**
```typescript

  setIsVisible(true);  // Trigger fade-in animation
}, []);
//  ‚Üë Empty array = run once on mount
```

**Lifecycle:**
```
1. Component mounts (first render)
   ‚Üì
2. isVisible = false (initial state)
   ‚Üì
3. Component renders (invisible)
   ‚Üì
4. useEffect runs
   ‚Üì
5. setIsVisible(true)
   ‚Üì
6. Component re-renders (now visible with animation!)
```

const validateForm = () => {
  const newErrors: {email?: string; password?: string} = {};

  if (!formData.email) {
    newErrors.email = 'Email is required';

  } else if (formData.password.length < 8) {

  }

  setErrors(newErrors);
  return Object.keys(newErrors).length === 0;
};

    setIsLoading(false);
  }
};
